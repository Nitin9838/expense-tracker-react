## ✨ Features

- Add income and expenses
- Delete transactions
- Real-time balance calculation
- Income vs Expense visualization (Pie Chart)
- Responsive UI
- Data persistence using LocalStorage

---

## 🛠 Tech Stack

- React.js
- CSS
- LocalStorage
- Chart.js (if you used it)

---

## 🧠 Concepts Used

- React Hooks (useState, useEffect)
- Component-based architecture
- State management
- Conditional rendering
- Data visualization

## ⚙️ How to Run

1️⃣ Clone the repository


git clone https://github.com/Nitin9838/expense-tracker.git

cd expense-tracker


2️⃣ Install dependencies


npm install


3️⃣ Run the project


npm run dev


4️⃣ Open in browser


http://localhost:5173
